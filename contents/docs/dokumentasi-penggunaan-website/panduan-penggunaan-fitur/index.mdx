
## 1. Pendaftaran dan Login Pengguna
        ### daftar Akun
          1. User akan diarahkan ke halaman login
          2. Klik tombol “register” di pojok bawah halaman.
          3. Isi form dengan data yang diminta, seperti :
             - username
             - Password
             - email
          4. Klik tombol "Daftar" untuk menyelesaikan pendaftaran.
          5. Anda akan menerima pemberitahuan 'registrasi berhasil' dan akan diarahkan ke halaman login.
        ### login akun 
          1. Isi form dengan data yang diminta, seperti:
             - username
             - Password
          2. klik tombol login dan akan diarahkan ke halaman utama
