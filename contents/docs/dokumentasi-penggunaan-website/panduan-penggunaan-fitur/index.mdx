
## 1. Pendaftaran dan Login Pengguna
        ### daftar Akun
          1. User akan diarahkan ke halaman login
          2. Klik tombol “register” di pojok bawah halaman.
          3. Isi form dengan data yang diminta, seperti :
             - username
             - Password
             - email
          4. Klik tombol "Daftar" untuk menyelesaikan pendaftaran.
          5. Anda akan menerima pemberitahuan 'registrasi berhasil' dan akan diarahkan ke halaman login.
        ### login akun 
          1. Isi form dengan data yang diminta, seperti:
             - username
             - Password
          2. klik tombol login dan akan diarahkan ke halaman utama

## 2. Penambahan data pelanggan
         ### Tambah pelanggan (customer)
            1. Masukkan nama pelanggan, No.hp, dan Alamat.
            2. Setelah memasukkan data pelanggan lalu klik simpan.
            3. Setelah mengklik simpan maka di halaman menu pelanggan mencul data pelanggan. 
            4. Pada data pelanggan dapat mengedit ulang data pelanggan apabila salah memasukkan Nama, No.hp, dan Alamat dan pada terdapat juga fitur hapus data pelanggan.

## 3. Penambahan transaksi pelanggan baru
         ### Transaksi baru
            1. Masukkan Nama pelanggan yang diambil dari data pelanggan.
            2. Masukkan Berat cucuian.
            3. Masukkan Tanggal  selesai.
            4. Masukkan jenis pakaian apa saja serta berapa banyak jumlah pakaian. 
            5. Setelah semuanya terisi maka klik simpan.
            6. Pada data transaksi terdapat fitur Invoice, Edit, dan Batalkan.

## 4. Laporan data pelanggan
         ### Filter Laporan
            1. Masukkan dari tanggal berapa ingin mencari data pelanggan.
            2. Masukkan sampai tanggal berapa ingin mencari data pelanggan.
            3. Setelah memasukkan dari tanggal berapa dan sampai tanggal berapa klik filter.
            4. Pada halaman filter laporan terdapat data laporan pelanggan dari tanggal berapa sampai tanggal berapa, klik cetak jika ingin mencetak data laporan.

## 5. Menu Pengaturan
         Dapat mengganti/mengubah berapa berat satuan tiap perkilo.

## 6. Menu Logout
         Untuk keluar dari akun Anda, klik tombol "Logout"
