## Tujuan

    1. Mempermudah pencatatan transaksi dan pesanan pelanggan.
    2. Mengurangi kesalahan manual dalam menghitung biaya dan mencatat data.
    3. Mempercepat proses pelayanan kepada pelanggan.

## permasalahan yang ingin diselesaikan

    1. Data pesanan sering tercampur atau hilang sehingga menyebabkan kesalahan seperti pakaian tertukar.
    2. Sulitnya melacak pemasukan, pengeluaran, dan laporan keuangan harian.
    3. Tidak ada catatan pelanggan tetap sehingga sulit memberikan layanan yang lebih personal, seperti promo atau diskon khusus.

## Teknologi Yang Digunakan

Proyek website kasir laundry ini menggunakan berbagai teknologi untuk memastikan fungsionalitas dan pengalaman pengguna yang optimal, antara lain :
    1. Bahasa Pemrograman: PHP dan JavaScript untuk pengembangan backend.
    2. Framework: Bootstrap untuk styling dan responsif
    3. Database: MySQL untuk menyimpan data.
    4. Frontend: HTML, CSS, dan JavaScript untuk membangun antarmuka pengguna yang responsif dan interaktif.
